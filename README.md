# Simple Node.js Web Server

## Description
This is a basic web server built using the native Node.js `http` module. It handles routing for a home page, about page, and contact page, and includes a custom 404 error page for invalid routes.

## Features
* **Asynchronous File Handling:** Uses `fs.readFile` to serve HTML files without blocking the server.
* **Custom Routing:** Directs users to `/home`, `/about`, or `/contact`.
* **Error Handling:** Sends a `404 Not Found` status code and a custom page for undefined routes.
* **CSS Styling:** All pages are styled for a clean, user-friendly look.

## How to Run
1. Clone the repository.
2. Open your terminal in the project folder.
3. Run the command: `node server.js`
4. Visit `http://localhost:3000/home` in your browser.
